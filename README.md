# 🌟 서울풍납초등학교 6학년 1반 칭찬 & 알림 보드

## 📖 프로젝트 소개
교사 개발자 해커톤 직무연수 산출물입니다.
우리 반 학생들의 따뜻한 칭찬과 학급 공지사항을 실시간으로 나누기 위해 제작된 학급 전용 웹 보드입니다.

## 🚀 주요 기능
- **안전한 접속:** 구글 계정 로그인을 통한 학급 구성원 인증 (Firebase Auth)
- **실시간 소통:** 새로고침 없이 즉시 반영되는 게시글 작성 (Firestore Database)
- **따뜻한 반응:** 게시글 내용 지우기, 학생 프로필 아바타, 좋아요/최고 이모티콘 및 댓글 기능
- **반응형 웹:** PC, 태블릿, 스마트폰 등 다양한 기기에 최적화된 화면 (Tailwind CSS)
